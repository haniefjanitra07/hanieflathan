How to Change PayPal Email:

To be able to receive payments, you need to have a PayPal account no matter its personal starter or business account. Any Fully approved PayPal account would work just fine. To Change the PayPal Email Address for receiving payments do as follows

    1. Go to Blogger >> Layout >> PayPal Email >> Edit
    2. In the text area enter your PayPal Email address
    3. Save the Widget, Press "Save" Button

How to Edit (Homepage) Slider

You can edit, add or delete slider slides with great ease using "Templateism Admin Panel"

    1. Go to Blogger >> Layout >> Slider >> Add a Gadget
    2. From the list Select "Image"
    3. Upload Slider Image, Add Title, Caption and Link if needed
    4. Save the Widget, Press "Save" Button

How to add (Homepage) Testimonials

You can edit, add or delete Testimonials with great ease using "Templateism Admin Panel"

    1. Go to Blogger >> Layout >> Testimonials >> Add a Gadget
    2. From the list Select "Image"
    3. Upload Testimonial image, Add Title, Caption and Link if needed
    4. Save the Widget, Press "Save" Button

How to Edit Newsletter Widget

The newsletter widget enables you to send automatic updates to your subscribers to their email inbox, its a great way to keep your users updated with your newly published articles.

    1. Go to Blogger >> Layout >> Newsletter >>Edit
    2. Enter Your Feedburner URL
    3. Save the Widget, Press "Save" Button

How to Add a Product?

To add a new product, you have to do the same thing that you usually do while publishing a normal article in blogger:

    1. Go to Blogger >> Add a New Post
    2. Enter title, Upload Product Image, etc
    3. Once everything is done, press "Publish" and your product is live ready to sell

How to Add Price to a Product

Every Thing has its price, so we have to put up price for our different products.

    1. Go to Blogger >> Add new Post >> Labels
    2. In Labels enter price_50$ (Where 50$ is the price of the product).
    3. Once everything is done, press "Publish" and your product is live ready to sell
